# Address-Book

The Address Book project is a desktop application designed for efficiently managing and storing personal contact information. It provides users with a user-friendly interface to add, view, edit, and delete contacts, including details such as:

Name: First name, last name
Phone Number: Multiple phone numbers per contact
Email Address: Email addresses associated with the contact
The application allows users to organize contacts by categories (e.g., family, friends, work), search for contacts by name or phone number, and export or import contact data. It aims to be an intuitive and simple solution for individuals who need to manage their personal or professional contacts efficiently.

Key Features
Add Contacts: Store essential information such as names, phone numbers, emails, and addresses.
Edit Contacts: Update or modify any contact details.
Delete Contacts: Remove contacts that are no longer needed.
Search Functionality: Quickly search for contacts by name, phone number, or email.
Contact Categories: Group contacts by predefined categories like family, friends, or work.
Import/Export: Import contacts from and export to common formats (e.g., CSV, vCard).
User Interface: Clean and simple desktop interface designed for ease of use.
